# Schematics — The Teller

## Wiring Diagram

Open `wiring-diagram.drawio` with [draw.io](https://app.diagrams.net) to view and edit the full wiring diagram.

For a text-based version of all connections, see `../wiring-diagram.md`.

## Exporting

To export as PNG or PDF from draw.io:
1. Open the `.drawio` file at https://app.diagrams.net
2. File → Export As → PNG or PDF
3. Save the exported file in this folder as `wiring-diagram.png` or `wiring-diagram.pdf`
