# Electronics Bill of Materials

| Component | Quantity | Notes |
|---|---:|---|
| Raspberry Pi 5 (main board) | 1 | Main embedded computer |
| USB sound card (audio I/O) | 1 | Provides microphone input and audio output |
| Hook switch (GPIO input) | 1 | Detects handset lift and hang-up state |
| RJ9 connector + handset wiring | 1 | Connects handset microphone and speaker |
| GPIO jumper wires | 1 set | Internal low-voltage wiring between switch and Pi |
| USB-C power supply | 1 | 5V power source for the Pi |
